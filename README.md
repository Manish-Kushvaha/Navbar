# 📌 Static Sidebar Navbar

A clean and responsive **sidebar navigation menu** built using pure HTML and CSS. This project demonstrates a modern UI layout with icons, sections, and a collapsible sidebar design.

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  

---

## ✨ Features

- 📱 Responsive sidebar design  
- 🎨 Clean and modern UI  
- 📂 Multiple navigation sections (Gallery, Shortcuts, Events, etc.)  
- ❌ Close button for sidebar  
- 🔗 Social media icons at bottom  
- ⚡ Lightweight (No JavaScript required)
